# 🛒 Django E-Commerce Store

A fully functional E-Commerce web application built with Django.

## 🚀 Features

- User Authentication (Login & Logout)
- Product Listing
- Add to Cart
- Shopping Cart
- Checkout
- Order Placement
- Admin Panel
- Responsive UI using Bootstrap

## 🛠 Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite
- Git
- GitHub
- Render
